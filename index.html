About Cuphead

Cuphead is a modern game with sort of vintage style graphics, it will give you an impression that it's another fun and easy game but it's not actually as it looks like (you're gonna rage a lot). It's the story of two Cup brothers who gambled with the devil and lost their souls so they have to get back at him. Cuphead is all about fighting bosses on different islands, and buy new gear with coins you earn by playing some special levels called "run and gun". It contains in total 3 main islands called Inkwell Isle one and two each one of them contains 5 boss levels and 2 run and gun levels, and Inkwell Isle three that contains 7 boss levels and 2 run and guns levels. The final island is called Inkwell Hell in contains a boss rush level and the final boss level.

Inkwell Island 1

Level Name: Botanic Panic!

One of the first unlocked bosses you can fight,it's quite easy and a great way to start the game.

Level Name: Ruse Of An Oose

This is a tricky one you should try to evade as much as possible (goodluck with that).

Level Name: Threatenin' Zeppelin

This one itroduces a new mechanic to the game and if you find yourself raging too much just quit.

Level Name: Clip Joint Calamity

This boss is mainly based on luck so just keep trying till you get the easiest waves.

Level Name: Floral Fury

This one's a bit tricky but a great boss the finish with.
